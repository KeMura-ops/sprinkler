## アプリケーションのURL
### **https://sprinkler-32865.herokuapp.com**

## 制作背景
私の身近なところにプロのイラストレーター志望の方が居るので、イラストレーターを目指している方、イラストを書き始めた初心者の方が気軽に作品を投稿して交流することが出来るSNSアプリケーションを作ろうと思い至り、現在作成途中の画像投稿アプリケーションとなります。
